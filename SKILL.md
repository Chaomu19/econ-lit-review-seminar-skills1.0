# 经济学文献综述组会｜Econ Literature Review Seminar Skills 1.0

## 目标

帮助用户把一组经济学论文整理成可用于组会汇报的文献综述材料，同时避免一开始就为每篇论文写冗长笔记。

默认流程：

1. 构建并核验论文池。
2. 将确认无误的 PDF 转换为结构可用的 Markdown。
3. 生成综述大纲、论文分组和阅读优先级。
4. 按板块逐步撰写 `PPT完整内容.md`。
5. 让用户只审核真正会出现在 PPT 中的文字、图表和论断。
6. 根据模板生成初版 PPT。
7. 对重复页型尽量用程序批量复用修改。
8. 只为可能被老师追问的论文准备兜底精读笔记。

## 核心规则

- 不上传或提交原始论文 PDF。
- 不默认认为自动化论文下载可靠。应让用户核验下载结果，或直接使用用户已确认的本地 PDF。
- 工作流按研究板块组织，不按固定天数组织。
- 除非用户明确要求，或某篇论文需要兜底精读，否则不要为每篇论文都创建完整的 `组会呈现.md`。
- 优先生成 `PPT完整内容.md`，再围绕实际展示的页面、图表、公式和论断引导用户阅读。
- 使用论文内容时，优先依据转换后的 Markdown 和引用标准，不凭空补充结论。
- 避免把流程绑定到个人 API、本机绝对路径或私人账号。
