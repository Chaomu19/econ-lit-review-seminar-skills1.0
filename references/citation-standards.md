# 引用与证据标准

## 核心标准

组会中的每个实质性论断都应基于以下来源之一：

- 转换后的论文 Markdown。
- 当 Markdown 转换不完整时，回查原始 PDF。
- 用户提供的阅读笔记。
- 明确标注为“基于上述材料的推断”的综合判断。

## 谨慎使用论文证据

每个与具体论文相关的论断，应尽量保留：

- 作者和年份。
- 研究场景、市场、国家或制度背景。
- 方法类型。
- 讨论原文图表时标注 `Fig.` 或 `Table` 编号。
- 该论断是直接结果、模型含义、政策解释，还是你的综合判断。

## 图表使用规则

为 PPT 选择图表时：

- 优先选择能直接支撑本页核心论断的图表。
- 尽可能记录原文 `Fig.` 或 `Table` 编号。
- 解释图表展示什么、关键模式是什么，以及它的经济学含义。
- 不要因为图表好看就纳入；图表必须服务论证。

## 不确定性标记

必要时使用明确标签：

- `原文结论`：论文直接说明或证明的内容。
- `基于模型的含义`：根据模型结构推导出的含义。
- `汇报综合判断`：跨论文综合后的判断。
- `需要复核`：图表编号、数值或论断仍需检查。

## 避免

- 编造公式或实证结果。
- 只根据摘要层面的说法进行过度延伸。
- 夸大外部有效性。
- 凭记忆使用未核验数字。
- 把 AI 总结当作原文结论展示。

## PPT 引用格式

幻灯片上使用简洁引用：

- `Author Year`
- `Author Year, Fig. X`
- `Author Year, Table X`
- `Author Year, model setup`

在笔记或 Markdown 草稿中保留更完整的来源上下文。
