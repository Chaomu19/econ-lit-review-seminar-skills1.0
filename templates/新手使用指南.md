# 新手使用指南

这份指南适合第一次使用本 skill 的用户。你不需要先理解所有文件，只要按下面步骤走。

## 一句话理解

这个 skill 不是帮你“逐篇抄论文摘要”，而是帮你把一批论文快速组织成一场能讲清楚的文献综述组会。

核心方法是：

```text
先确定论文池和综述主线
-> 再把 PDF 转成 Markdown
-> 再让 AI 自动分组和判断精讲层级
-> 直接生成 PPT完整内容.md
-> 围绕 PPT 展示内容反向阅读
-> 最后只给可能被追问的论文补深度笔记
```

## 第 0 步：准备材料

你需要准备：

- 一个组会主题。
- 一批候选论文，可以很粗糙。
- 已确认正确的论文 PDF。
- 一个能把 PDF 转成 Markdown 的工具。

注意：不要完全依赖 AI 自动下载论文。自动下载很容易下载错版本、错论文或无关文件。

## 第 1 步：整理论文池

复制 [paper-pool-template.md](paper-pool-template.md)，填入候选论文。

你可以这样问 AI：

```md
我准备做一个关于【主题】的经济学文献综述组会。
下面是我的候选论文池。请你帮我检查：
1. 这些论文是否覆盖了这个主题的核心问题；
2. 是否遗漏经典文献或关键近期文献；
3. 是否有明显不相关或重复的论文；
4. 初步判断哪些适合精讲、重点阅读、补充阅读。
```

建议让 2-3 个模型分别审阅一次，然后由你人工做最后判断。

## 第 2 步：PDF 转 Markdown

把确认无误的 PDF 放进同一个本地文件夹，再转换成 Markdown。

转换后使用 [pdf-to-md-checklist.md](pdf-to-md-checklist.md) 检查：

- PDF 是否正确。
- 题名、作者、年份是否正确。
- 章节结构是否完整。
- 关键公式是否可读。
- 图表和图注是否能找到。

如果转换质量很差，不要直接让 AI 依赖这份 Markdown 生成关键论断。

## 第 3 步：让 AI 分组和判断优先级

把转换后的 Markdown 目录告诉 AI，让它先做结构设计。

你可以这样问：

```md
请根据这些转换后的论文 Markdown，帮我完成组会综述结构设计：
1. 提出 4-6 个研究板块；
2. 每个板块说明核心研究问题；
3. 判断每篇论文属于精讲、重点阅读还是补充阅读；
4. 给出推荐 PPT 页序；
5. 标记哪些论文可能需要兜底精读。
```

这一阶段不要急着写每篇论文长笔记。先把综述结构定住。

## 第 4 步：生成 `PPT完整内容.md`

复制 [ppt-complete-content-template.md](ppt-complete-content-template.md)，按板块逐步生成。

推荐每次只让 AI 生成一个板块，例如：

```md
请根据已确认的论文分组和引用标准，先生成“市场设计与投资激励”这个板块的 PPT完整内容。
要求：
1. 包含板块首页、板块总览表、精讲论文页；
2. 每页写清页面标题、核心内容、本页关键判断；
3. 使用图表时标出原文 Fig. 或 Table 编号；
4. 不确定的内容标记“需要复核”。
```

这样做可以避免上下文太长，也方便你边审边改。

## 第 5 步：人工审核和高效阅读

你不需要先完整精读所有论文。

优先阅读：

- PPT 中会展示的图。
- PPT 中会展示的表。
- 精讲论文的模型设定。
- 反直觉或可能引发争议的结论。
- 你自己讲不顺的页面。

审核时重点删改：

- 太长的文字。
- 没有服务主线的图表。
- AI 说得太满但证据不足的判断。
- 你自己不能解释清楚的结论。

## 第 6 步：生成 PPT

如果你有 PPT 模板，可以让 AI 根据 `PPT完整内容.md` 生成初版 PPT。

建议先生成初版，再人工修改 1-2 页代表性页面。文献综述 PPT 通常有大量重复页型，后续可以让 AI 用代码把你修改好的页面样式应用到同类页面。

## 第 7 步：准备兜底精读

只有这些论文需要兜底精读：

- PPT 中用了 2-3 页精讲的论文。
- PPT 中展示了关键公式或核心图表的论文。
- 结论反直觉、容易被老师追问的论文。
- 你自己还讲不透的论文。

使用 [deep-reading-note-template.md](deep-reading-note-template.md) 生成笔记即可。

## 最常见错误

- 一开始就为每篇论文写长篇笔记。
- 没核验 PDF 是否正确就让 AI 总结。
- 把所有论文都标成精讲。
- PPT 里塞太多图表，但自己讲不清楚。
- 不区分原文结论、模型含义和自己的综合判断。

## 最推荐的最终成果

一次完整使用后，你最好得到：

- `论文池表.md`
- `PDF转Markdown检查清单.md`
- `论文分组与优先级表.md`
- `PPT完整内容.md`
- `初版PPT.pptx`
- 若干篇 `兜底精读笔记.md`
